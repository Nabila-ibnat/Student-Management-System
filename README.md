# Student Management System

## Project Overview

This is a command-line based Student Management System built using Python.  
It allows users to add, view, search, update, and delete student records using file handling.

---

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Grade Calculation

---

## Data Storage

Stored in:
```
Data/students.txt
```

Format:
```
Name,Age,Marks
```

Example:
```
John,20,85
Alice,22,78
```
